Angular new control flows with new syntax:

@if - @else
Angularda döngü ve koşul yazmanın yeni şekli diyebiliriz. *ngFor *ngif gibi `structural directive`ler hala kullanılabiliyor. Bu yeni gelen yapılar sadece yeni kullanıcı dostu-“beginner-friendly” hale getiriyor.


@if (sinyal()) {
Gel
} @else {
Git
}

@switch - @case


@switch (sinyal()) { @case (true) { Doğru } @case (false) { Yanlış } }

} @else {

Tek Sayı

}
